Instalar o Python, caso não esteja instalado, correndo o instalador contido nesta pen:
- python-2.7.10.msi
Não alterar o path predefinido para a instalação do Python (C:\Python27)


Correr o thescript.bat, caso possível como Administrador.
Vai ser aberta a linha de comandos,onde vai aparecer 
uma lista das drives no sistema, para que possamos saber
em qual delas está inserida a pen. Também é possível saber a letra da drive
olhando para a pasta "O meu computador".
 Será pedido para inserir a drive correspondente à da pen, na linha de 
comandos (em princípio D,E, F ou G). Insira apenas a letra correspondente 
(por exemplo, inserir apenas F).
É depois pedido para inserir a versão do PostgreSQL instalada no Computador.
É possível verificar a mesma em C:\Program Files\PostgreSQL\

Antes de correr o pgtune, vai ser feito um backup da configuração existente em
C:\Program Files\PostgreSQL\X.X\data\postgresql.conf.backup
Caso seja a primeira vez que corre o script na máquina e esse ficheiro ainda não
exista, o programa pergunta se o destino seleccionado é um ficheiro (F) ou
um directório (D). Escolher a opção ficheiro (F).
